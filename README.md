DigiBlink
=========

Digispark LibUSB RGB control program (based on send.cpp from the official example)


Usage:
======

    blink [color]

color can be:
"red", "green", "blue", "white", "gray", "grey", "yellow", "pink", "cyan", "black" or "off"

or you can use RGB values:

    blink [red] [green] [blue]

* red = 0 - 255
* green = 0 - 255
* blue = 0 - 255
